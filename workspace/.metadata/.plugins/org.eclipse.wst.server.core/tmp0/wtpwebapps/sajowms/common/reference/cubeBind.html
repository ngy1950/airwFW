  • CubeBind

    데이터 처리를 위한 객체

  • CubeBind.cubeDataGridMap

    CubeMap. grid 객체를 저장

  • CubeBind.cubeDataGridRowMap

    CubeMap. grid에 bind될 row객체를 저장

  • CubeBind.cubeDataGridPagingMap

    CubeMap. grid에 연결된 paging객체를 저장

  • CubeBind.cubeDataGridPageinfoMap

    CubeMap. grid에 연결된 page info객체를 저장

  • CubeBind.comboScan(areaId)

    combo 자동완성 기능 적용

  • CubeBind.searchComboScan(tmpRowHtml)

    데이터 변화에 따라 새로운 combo data를 메핑해야 하는 경우 사용.
    cubeDataSearchCombo="모듈,sql id,request param id" - VALUE_COL, TEXT_COL
    request param id 객체의 value를 추출하여 CODE키로 전송

  • CubeBind.gridScan(areaId)

    areaId 내에 포함된 grid를 찾아서 사용 가능하도록 준비시킴

  • CubeBind.gridInfoSave($grid, rowAttr)

    $grid를 grid로 사용 가능하도록 준비시킴
    rowAttr - default cubeConfig.DATA_ROW_ATT

  • CubeBind.gridRowPrintNum(gridId, num)

    gridId의 grid에 출력된 row의 개수를 저장한다.
    row추가시 첫 row인지 판단하는 근거가 된다.

  • CubeBind.gridNewRow(gridId)

    gridId에 새로운 row를 추가해준다.

  • CubeBind.gridAddRow(gridId)

    gridId에 row를 추가한다.

  • CubeBind.gridRowCopy(copyBtn, gridId)

    copyBtn이 포함된 row를 복사해서 추가해준다.

  • CubeBind.gridBind(gridId, data, editGridType, groupCount, appendType)

    gridId에 data를 bind한다.
    editGridType - grid가 edit가능한지 설정한다.
    groupCount - row obj가 data의 여러 행을 처리하는 경우 개수를 설정한다.
    appendType - true인 경우 기존 데이터를 삭제하지 않고 계속 추가한다.

  • CubeBind.bindGridBeforeView($dataRow, createType)

    edit grid인 경우 row들의 데이터 화면 표시를 위해 처리해준다.
    new row인 경우 createType을 true로 지정한다.

  • CubeBind.gridRowStateChange($row, stateAttr)

    해당 row의 상태를 stateAttr값으로 변경해 준다. CRUD

  • CubeBind.gridRowDelete(deleteBtn)

    해당 row를 삭제해준다.

  • CubeBind.dataBind(data, areaId)

    data를 areaId내이 객체 id들과 bind한다.

  • CubeBind.dataNameBind(map, areaId, tail)

    map에 있는 데이터를 areaId내이 객체 name들과 bind한다.
    name의 중복등으로 구분을 위한 tail이 붙어 있는 경우 key+tail로 name을 찾아 bind한다.

  • CubeBind.dataIdBind(map, areaId, tail)

    map에 있는 데이터를 areaId내이 객체 id들과 bind한다.
    id의 중복등으로 구분을 위한 tail이 붙어 있는 경우 key+tail로 id를 찾아 bind한다.

  • CubeBind.dataClear(areaId)

    areaId에 있는 입력값들을 삭제한다.

  • CubeBind.gridPageInfoBind(gridId)

    grid의 paging정보를 해당 gird를 바라보고 있는 grid info에 적용해준다.

  • CubeBind.gridPageCountChange(gridId, count)

    gird의 page count(한페이지당 출력 개수)를 변경한다.

  • CubeBind.gridPageCountReset(gridId, count)

    gird의 page count(한페이지당 출력 개수)를 설정한다.

  • CubeBind.gridPageBind(gridId, data)

    grid paging을 만들어 준다.

  • CubeBind.gridPageData(gridId, map)

    grid paging을 위한 정보를 map에 담아 리턴해준다.

  • CubeBind.gridPageChange(gridId, pageNum)

    grid page를 변경해준다.

  • CubeBind.sendGridParamData(gridId, map)

    edit grid에 데이터를 추출해 map에 담아준다.

  • CubeBind.sendParamData(map, areaId)

    데이터 전송을 위해 areaId에 있는 데이터를 map에 저장해준다.

  • CubeBind.paramDataFilter(map, $dataRange)

    $dataRange에 있는 입력값들을 map에 저장해준다.

  • CubeBind.selectBind(selectId, data)

    combo에 data를 자동으로 셋팅해준다. VAL_COL, TEXT_COL

  • CubeBind.gridExcelRequest(url, map, gridId)

    excel file download를 요청한다.
    url로 map데이터를 통한 요청을 하며, excel file생성을 위한 정보를 gridId내에서 가져온다.

  • CubeBind.calendarBind(calendarId, data)

    fullCalendar에 데이터를 셋팅해준다.

  • CubeBind.calendarDataRemove(calendarId, eventObj)

    fullCalendar에 데이터를 삭제해준다.

  • CubeBind.calendarNav(calendarId, type)

    fullCalendar에서 nav버튼 동작을 처리한다.

  • CubeBind.calendarDateSet(calendarId)

    변경된 fullCalendar에서 날자를 표시해준다.

  • CubeBind.dateSet(dateId, type, num)

    dateId에서 연월일을 변경해준다.

  • CubeBind.toString()

    return "CubeBind"
