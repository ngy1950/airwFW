  • CubeBoard

    게시판 컨트롤에 필요한 작업을 처리한다.

  • CubeBoard.checkBoard($obj)

    게시판으로 설정된 객체를 적용해준다.

  • CubeBoard.boardGridScan($scanArea)

    게시판에 포함된 grid를 셋팅한다.

  • CubeBoard.selectBoard(itemObj, boardType, contentId)

    board내 특정 item을 이용해 board box에 저장된 속성으로 간단하게 view board를 실행한다.

  • CubeBoard.selectBoardBox(boardboxId, boardType)

    board box id를 이용해 board box에 저장된 속성으로 간단하게 view board를 실행한다.

  • CubeBoard.viewBoard($boardBox, boardCode, boardType, contentId, listType)

    선택한 게시판을 보여준다.

  • CubeBoard.getContentList(boardCode, boardListId)

    게시글 리스트를 가져온다.

  • CubeBoard.getContent(contentId, boardBoxId)

    게시글을 가져온다.

  • CubeBoard.getContentFileList(contentId, $boardBox)

    게시글에 포함된 파일 리스트를 가져온다.

  • CubeBoard.getContentCommentList($boardBox)

    게시글에 포함된 덧글 리스트를 가져온다.

  • CubeBoard.insertContentSuccess(json, boardBoxId)

    게시글을 저장후 조회 화면으로 돌아간다.

  • CubeBoard.updateContentSuccess(json, boardBoxId)

    게시글을 수정한 후 조회 화면으로 돌아간다.

  • CubeBoard.deleteContent(itemObj)

    게시글을 삭제합니다.

  • CubeBoard.insertComment(commentBoxId)

    게시물에 덧글을 추가합니다.

  • CubeBoard.addFileForm(fileListId)

    게시글에 파일 입력 폼 추가

  • CubeBoard.fileFormRemove(btnObj)

    게시글에 파일 입력 폼 삭제

  • CubeBoard.contentFileDelete(btnObj, uuid)

    게시글에 파일 삭제

  • CubeBoard.contentCommentDelete(btnObj, commentId)

    게시글 덧글 삭제
