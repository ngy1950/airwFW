  • CubeCommon

  • CubeCommon.loadingViewType

    Boolean. ajax 호출시 자동으로 표시되는 loading의 표시 유무.

  • CubeCommon.cubeLoadingCount

    ajax의 비동기 호출 시 여러건의 call back을 감지하여 모두 완료되어야ㅣloading을 해지한다.

  • CubeCommon.$cubeFwAjaxIndicator

    loading시 화면을 블락한다.

  • CubeCommon.$cubeFwIndicatorDiv

    loading이미지가 표시되는 영역

  • CubeCommon.$cubeFwUploadText

    ajax fileupload시 진행상태가 표시되는 영역

  • CubeCommon.displayLoadingSet()

    loaging을 표시하기 위한 기본 영역을 생성한다.

  • CubeCommon.loadingViewOff()

    logding을 사용하지 않도록 설정

  • CubeCommon.loadingViewOn()

    loading을 사용하도록 설정

  • CubeCommon.displayLoading(type)

    loading을 표시한다. type - true:표시, false:해지

  • CubeCommon.commonLoad(vId)

    최초 loading 준비

  • CubeCommon.enterKeyCheck(event, fncName)

    input에서 enter key조작시 특정 함수 호출

  • CubeCommon.setElementValue($obj, data)

    $obj jQuery객체에서 값을 셋팅한다.

  • CubeCommon.getElementValue($obj)

    $obj jQuery객체에 값을 추출한다.

  • CubeCommon.getArea(vId)

    vId에 해당하는 jQuery객체를 가져온다. 지정 영역이 없는 경우 body전체를 영역으로 잡는다.

  • CubeCommon.getObjType(obj)

    obj의 타입을 가져온다.

  • CubeCommon.getJObj(obj)

    obj의 값이 어떤 형태이건 jQuery객체로 반환한다.

  • CubeCommon.getObjId(obj)

    지정한 obj의 id를 반환한다.

  • CubeCommon.isEmpty(obj)

    obj가 어떤 형태이건 값이 없는지 판단한다.

  • CubeCommon.isNotEmpty(obj)

    obj가 어떤 형태이건 값이 있는지 판단한다.

  • CubeCommon.NVL(obj, val)

    obj가 Empty이면 지정한 val을 리턴한다.

  • CubeCommon.popupOpen(url, width, height)

    popup을 open한다.

  • CubeCommon.msg(txt)

    해당 txt를 화면에 보여준다.

  • CubeCommon.leftPadding(data, char, len)

    data의 길이가 len보다 작으면 왼쪽부터 char를 채운다.

  • CubeCommon.fillString(char, len)

    char를 len개수만큼 만들어서 리턴한다.

  • CubeCommon.replaceAll(str, char, nChar)

    str에 포함된 char를 nChar로 변경한다.

  • CubeCommon.toString()

    return "CubeCommon"
