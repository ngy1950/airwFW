  • CubeGridRow(obj)

    grid에서 선택된 row의 정보를 추출하기 위한 객체. 인스턴스화 시킨 후 사용.

  • CubeGridRow.$rowObj

    obj이 포함된 row객체를 찾는다.

  • CubeGridRow.data

    CubeMap. row에 포함된 데이터 값들은 name을 key로 data에 put한다.
    select의 경우 value와 text를 함께 추출하기 위해 구분한다.
    text는 select의 name에 cubeConfig.DATA_COMBO_SELECTED_TEXT_TAIL를 붙여 키로 저장한다.

  • CubeGridRow.getRowStates()

    해당 row의 상태를 리턴한다. CRUD
