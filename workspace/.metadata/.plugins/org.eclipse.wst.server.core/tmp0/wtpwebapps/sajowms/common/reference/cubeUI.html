  • CubeUI

  • CubeUI.typeCheck($obj)

    UI로 설정된 객체를 UI로 지정해준다.

  • CubeUI.resetEditor(editorId)

    web editor로 지정했던 객체를 초기화 해준다.

  • CubeUI.checkEditor($obj)

    web editor를 지정해준다.

  • CubeUI.checkCalendar(data, $obj)

    full calendar를 지정해준다.
    data - 초기 데이터값

  • CubeUI.checkTree($obj)

    tree 지정

  • CubeUI.loadTree($tObj, selectId)

    $tObj를 tree로 만들어준다.
    특정 노드를 열린 상태로 만들기 위해 노드의 id를 selectId로 전달한다.

  • CubeUI.colGrouping(tableId, colNum)

    table의 colNum에 해당하는 td중 연속해서 같은 값을 가진 경우 rowspan을 해준다.

  • CubeUI.dateFormat(date, format)

    날자 형식에 맞춰서 출력해준다.

  • CubeUI.tabOpen(tabId, index)

    tabId에서 선택산 index를 open해준다.

  • CubeUI.zeroFormat()

    return "CubeUI"

  • CubeUI.toString()

    return "CubeUI"
