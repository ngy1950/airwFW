FileDownload

Constructors

 

 

FileDownload

FileDownload 객체를 생성합니다.

 

Syntax

    FileDownload ( HttpServletRequest request, HttpServletResponse response)
    FileDownload ( HttpServletRequest request, HttpServletResponse response, String charset)

Parameters

request

[in] HttpServletRequest 객체를 입력합니다.

response

[in] HttpServletResponse 객체를 입력합니다.

charset

[in] UTF-8 환경이 아닐 경우, EUC-KR 등 시스템에 맞는 Charset을 입력합니다.

 

Return Values

 

Remarks

NamoCrossUploader Server Java Edition은 기본적으로 UTF-8(다국어) 환경에서 동작합니다. charset은 UTF-8 환경이 아닐 경우만 입력해 주십시오.

 

Sample Codes

Java

FileDownload fileDownload = new FileDownload(request, response);
