FileUpload

Methods

 

 

deleteUploadedFiles

업로드 된 파일을 모두 삭제합니다.

 

Syntax

    void deleteUploadedFiles ()

Parameters

 

Return Values

 

Remarks

업로드 중 예외가 발생했을 경우는 내부적으로 업로드 중인 임시 파일들을 모두 삭제합니다.
deleteUploadedFiles() Method는 업로드 외 로직에서 예외가 발생했을 경우 불필요한 데이터를 삭제하기 위한 방법을 제공합니다.
개별 파일을 선택적으로 삭제하기 위해서는 FileItem 객체의 deleteFile Method를 참조해 주십시오.

 

Sample Codes

Java

FileUpload fileUpload = new FileUpload(request, response);
try {	
	fileUpload.setAutoMakeDirs(true);
	String saveDirPath = request.getRealPath("/");
	saveDirPath += ("UploadDir" + File.separator);
	fileUpload.startUpload(saveDirPath); 	
}
catch(CrossUploaderException ex) { 
}
catch(Exception ex) { 
	// 업로드 외 로직에서 예외 발생 시 이미 업로드 된 파일을 삭제하는 예제입니다.
	fileUpload.deleteUploadedFiles(); 
} 
finally { 
	fileUpload.clear(); 
}
