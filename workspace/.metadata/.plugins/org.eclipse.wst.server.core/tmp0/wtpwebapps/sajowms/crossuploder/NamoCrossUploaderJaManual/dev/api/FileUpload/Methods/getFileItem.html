FileUpload

Methods

 

 

getFileItem

HTTP 요청 파라미터의 파일 데이터 정보를 FileItem 타입으로 반환합니다.

 

Syntax

    FileItem getFileItem ( String name )

Parameters

name

[in] HTTP 요청 파라미터의 name을 입력합니다.

 

Return Values

파일 데이터의 정보를 FileItem 타입으로 반환합니다. name에 해당하는 파일 데이터가 없을 경우 null을 반환합니다.
