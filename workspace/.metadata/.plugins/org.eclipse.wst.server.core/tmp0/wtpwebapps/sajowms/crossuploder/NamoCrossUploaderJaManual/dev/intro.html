소개

Namo CrossUploader Server Java Edition은

HTTP 프로토콜 기반의 Server-Side 업로드 컴포넌트로서 IE, Chrome 및 Firefox 등의 웹브라우저와 RFC1867 표준을 지원하는 서드파티 업로드 컴포넌트를 지원합니다.

리눅스, 유닉스 및 윈도우 시스템의 웹서버로 호스팅되는 모든 서블릿 환경에서 사용할 수 있습니다.

이 안내서는

개발자 안내서에서는 Namo CrossUploader Server Java Edition의 기능과 프로그램 제어를 위한 API(Application Programming Interface)에 대하여 안내합니다.
