Namo CrossUploader Server Java Edition Samples
Namo CrossUploader Server Java Edition 제품은 기본적으로 다국어(UTF-8) 기반에서 동작합니다.
인증 설명
기술지원 방법 등 설명
업로드 샘플
기본적인 파일 업로드 가장 기본적인 파일 업로드 예제입니다.
멀티플 파일 업로드 동일한 file field의 name 값으로 2개 이상의 파일을 업로드하는 예제입니다.
폼 데이터 업로드 파일 및 다양한 폼 데이터를 업로드하는 예제입니다.
Charset 지정 EUC-KR 등 특정 언어에 종속적인 환경에서 업로드하는 예제입니다.
파일 업로드 제한 파일 확장자, 전체 Content-Length 및 개별 파일 크기를 제한하는 예제입니다.
파일 복사 업로드 후 여러 경로로 파일을 복사하는 예제입니다.
예외 처리 업로드 외 로직에서 예외 발생 시 예외 처리 및 이미 업로드 된 파일을 삭제하는 예제입니다.
다운로드 샘플
기본적인 파일 다운로드 가장 기본적인 파일 다운로드 예제입니다. 파일 종류 및 웹브라우저의 옵션에 따라 웹브라우저에서 파일이 열릴 수 있습니다.
저장 모드 파일 다운로드 파일 종류에 관계 없이 항상 파일로 저장할 수 있는 파일 다운로드 예제입니다.
이어 받기 모드 파일 다운로드 이어받기가 가능한 파일 다운로드 예제입니다. 클라이언트에서 이어받기 요청이 있어야 이어받기로 동작합니다.
이미지 파일 처리
기본적인 이미지 정보 확인 기본적인 이미지 파일의 정보를 가져오는 예제입니다.
이미지 메타데이터 정보 확인 이미지 파일의 메타데이터 정보를 가져오는 예제입니다.
이미지 크기 수정 이미지 파일의 크기를 변경하는 예제입니다. (Thumbnail)
이미지 워터마크 이미지 타입의 마크를 원본 이미지에 삽입하는 워터마크 예제입니다.
텍스트 워터마크 텍스트 타입의 마크를 원본 이미지에 삽입하는 워터마크 예제입니다.
이미지 포맷 변경 이미지 파일의 포맷을 변경하는 예제입니다.
지원되는 이미지 포맷 확인 현재 시스템에서 읽기 및 쓰기가 가능한 이미지 포맷 이름 및 이미지 마임 타입을 확인합니다.
기타 샘플
BLOB 형식의 파일 업로드 & 파일 다운로드 업로드 한 파일을 BLOB 타입으로 DB에 저장하고, 다운로드하는 예제입니다.
